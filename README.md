## Science Journal of Hai Phong University

### Cài đặt
Backup `Science.bak`
Đổi chuỗi kết nối tại `Web.config`

### Chức năng
1. Lưu trữ bài báo
2. Tin tức
3. Tìm kiếm
4. Thống kê
...

### Quyền
1. Thành viên
2. Biên tập viên
3. Tổng Biên tập
4. Quản trị viên

### Công cụ
1. Visual Studio 2017
2. SQL Server 2016

### Công Nghệ
1. ASP.NET MVC 5
2. Entity Framwork v6.1.3
3. LinQ, ADO.NET, Database First
